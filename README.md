# Raghav Narang - Portfolio

Personal portfolio website showcasing my work as a Senior Software Engineer specializing in React, Next.js, and modern web development.

## Tech Stack

- **Framework:** Next.js 16.1.1 with App Router
- **UI:** shadcn/ui v2 with Tailwind CSS v4
- **Icons:** Phosphor Icons
- **Theme:** Dark mode with amber accent
- **Deployment:** Vercel

## Features

- Responsive design optimized for mobile and desktop
- SEO optimized with OpenGraph and Twitter cards
- Downloadable PDF resume
- Skills highlighting with visual indicators
- Timeline-based work experience
- Open source contributions showcase

## Development

```bash
npm install
npm run dev
```
